What is Web Coding

Web coding, is the 3 coding languages, that make up the web. Web coding is made up of Hyper Text Markup Language(html), Cascading Stylesheets(css), and JavaScript(js). This website talks about the 3 languages, and what they do. HTML is the content of a website, CSS is the decoration, and JS is the dynamics and user-input. These 3 languages are the languages you need to know for a computer programming career. But, even if your not old enough to get a career, it is not too early to learn.
