The code above, is a basic html layout, make a new file named index.html, and put it in a new folder named "Learn1ng WebC0de." Then, add 2 more new files names styles.css, and script.js inside the same folder. Copy the code in the image into index.html, in the rest of this lesson, you will learn what each line of code means.





The !DOCTYPE html code in this code tells the document it is the newest version of html. The html and /html tags start and end the page. Tags or elements are the things inside the <>. End tags have a "/" in the beginning of the tag to represent an end of an element.



The ... in the middle, just represents, that there is more code in the middle.





The Head Tag has everything that will run before all the other elements, yu]ou put elements like these in the head tag. The title element names the document. The Link and Script elements link our css and js to our html file.





Our body element contains all our content that is not in the head element. The h1 element stands for Heading One, and the p element stands for paragraph
